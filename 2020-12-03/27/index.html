理解 linux unix 的历史

发表于 2020-12-03
  • 北风承欢
首页 Linux 学习

1.unix 是 B 语言和 C 语言写出来的

2.linux 是基于 unix 写出来的

3. 目前世界的 linux 的主流的发行版有 ubuntu centos（红帽公司）

4.window 的安全性没有 Linux 高（原因是因为 window 的代码不是开源而 linux 是开源的 Linux 的用户发现了漏洞可以去反馈但是 window 不同因为他是不开源的所以就算是发现了漏洞也不敢去反馈因为你的源代码获取途径是违法的）所以 window 安全性不高而且还收费，Linux 安全性高但是生态没有 window 好。

5. 桥链接会导致 linux 和其他系统通讯但是可能造成 IP 地址冲突

6. 用 ANT 网络地址转换来访问外网，且 IP 地址不冲突

7. 主机模式：linux 是独立系统不能访问外网。
