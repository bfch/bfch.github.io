linux （网络协议）

发表于 2020-12-03
  • 北风承欢
首页 Linux 学习
  1. 物理层：电脑连接网络需要用光缆、电缆、双绞线、无线电波等方式连接网络这个就是物理层用物理手段连接电脑的。（是负责传送 0~1 的电信号）。
  2. 链接层：单纯的 0 和 1 没有意义所以 “链接层的功能” 在物理层的上方确定了 0 和 1 的分组方式。所以出现了一个就以太网的协议占据了主导地位，以太网规定一组电信号构成一个数据包叫做 “帧”（Frame）。每一个帧分成了 2 个部分一个标头和一个数据（他们加起来加叫 “帧”）
  3. 标头：存放的是发送者、接收者、数据类型的信息等等。（标头的长度固定为 18 字节）
  4. 数据：是数据包的具体内容。（"数据" 的长度，最短为 46 字节，最长为 1500 字节。因此，整个 "帧" 最短为 64 字节，最长为 1518 字节。如果数据很长，就必须分割成多个帧进行发送。）
