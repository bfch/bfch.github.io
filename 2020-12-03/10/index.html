MySQL 条件查询

发表于 2020-12-03
  • 北风承欢
首页 MySQL 学习

一、简单条件查询
（1）在学生表中查询出王琳的记录；

（2）在学生表中查询出计算机专业的学生记录；

（3）在学生表中查询出 1990 年以前出生的学生的姓名和出生日期；

（4）在学生表中查询出总学分不足 50 分的学生的姓名、专业和总学分

二、多条件查询
可以使用 and (&&)（而且）和 or（或）设置多条件查询。
例：查询出学生表中计算机专业的女生。

三、使用 between ……and (在…… 之间) 指定查询范围
Not between ……and 不在…… 之间 或 除 …… 之外
例：在成绩表中查询出成绩在 80 到 90 之间的记录

四、使用 in 指定查询范围
In（ ）在集合之内，not in ( ) 不在集合内
例：在学生表中查询出李方方、严红、王敏、李红庆这四个同学的情况

比较： select * from xs where xs_name like "李_";
