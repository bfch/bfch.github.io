一、变量的基本概念
1. 变量是用来存储数据的。
2. 变量是存在名字的。
3. 变量是通过名字来访问数据的。
4. 变量也是可以改变数据的。

二、变量的使用
1. 定义：可以在系统中自定义变量的名字，可以增加变量。
2. 赋值：可以在自定义变量的同时给变量赋值。
3. 可以通过变量的名字访问变量里面存储的数据。
4. 可以把变量从存储中删除。
